## :warning: Instalação <a name="install"></a>

```bash
# Instalar as dependências necessárias:
$ yarn

# Configurar as variáveis de ambiente:
$ .env.example to .env.local

# Iniciar o projeto:
$ yarn start

# Regras do Firebase:
$ database.rules.json
```
